About This Website

This is a website that contains information of my game, pixel art, scenterverse lore, etc. This is a passion project that I hope to build up so that I can make hopefully get a career in Web development one day.

Why create this?

I have created this website as a way to try out web development and to make keeping track of other things easier. For example I have always wanted to create a lore for my game and by making a website it allows me to do two important things at once. Another example is to show off the pixel art I have created and archive it.
